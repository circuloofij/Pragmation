circuloOfij_timeOperator  
==========================  
  
Operator para recoger tiempos para el Mashup en el Lab de Fi-Ware.  
  
Este componente sirve para injectar información en el widget Flotr Graph, para mostrar graficos relativos a los tiempos de respuesta de los pacientes.
